# TwinCAT .gitignore — AU-tomation standard

Canonical `.gitignore` for all AU-tomation TwinCAT 3 library and application repositories.

## Usage

Copy `.gitignore` to the root of the repository. No modifications are needed for standard TwinCAT PLC projects.

```powershell
Copy-Item "path\to\TwinCAT-Gitignore\.gitignore" "C:\TwinCAT projects\AUT_MyLib\.gitignore"
```

The `New-AuRepo.ps1` script in [New-GitHub-Repository](../New-GitHub-Repository/) should also be updated to reference this file when creating new repositories.

## What is ignored

| Section | Patterns | Notes |
|---|---|---|
| TwinCAT PLC generated | `*.tpy`, `*.library`, `*.compileinfo`, `LineIDs.dbg`, … | Rebuilt on every compile |
| TwinCAT C++ / shared types | `*.tmc`, `*.tmcRefac` | Remove `*.tmc` if the project uses shared types, C++, or TcCom modules |
| Project backup and temp files | `*.tsproj.bak`, `*.xti.bk?`, `*.xt?.orig`, … | Backup copies created by TwinCAT XAE |
| TwinCAT Multiuser | `**/.TcGit/` | Multiuser merge metadata |
| Generated folders | `_Boot/`, `_CompileInfo/`, `_Libraries/`, `_Logs/`, `_ModuleInstall/`, `_Deployment/`, `_Repository/` | All regenerated by TwinCAT |
| Visual Studio | `.vs/`, `*.suo`, `*.user`, `*.userosscache`, … | Local IDE state |
| CI build artifacts | `*.sarif.json`, `TcUnit_Results.xml`, `build.log` | Generated by TcCIBuilder / GitHub Actions; not part of source |
| Claude Code | `CLAUDE.md`, `.claude/` | Local AI assistant settings |
| Python cache | `__pycache__/`, `*.pyc` | From `generate_manifest.py` if run locally |

## Updating

When a new ignore rule is needed across all repos:

1. Edit `.gitignore` in this folder (single source of truth).
2. Copy the updated file to every `AUT_...` repository root.
3. Commit and push each repo.
